- **DQN** (`agents/dqn_agent.py`)
- **Monte Carlo** (`agents/monte_carlo_agent.py`)
- **Q-Learning** (`agents/q_learning_agent.py`)
- Other files in `environment/` define or customize the Pac-Man environment.
